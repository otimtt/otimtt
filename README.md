## Featured Project

### [Vampire Of Date (VOD)](https://github.com/otimtt/vampire-of-date)

Terminal-based defensive audit tool for inspecting the local host and Android devices connected through USB/`adb`.

Main points:

- Interactive terminal interface with a home screen
- Host audit with platform-aware collectors
- Android security checks through `adb`
- Findings organized with severity, evidence, and recommendation
